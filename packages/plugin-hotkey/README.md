# @easy-editor/plugin-hotkey

Hotkey plugin for EasyEditor.

## Hotkeys

| Action | Shortcut |
|--------|----------|
| Undo | ⌘ + Z / Ctrl + Z |
| Redo | ⌘ + Y / Ctrl + Y |
| Lock/Unlock | ⌘ + Shift + L / Ctrl + Shift + L |
| Show/Hide | ⌘ + Shift + H / Ctrl + Shift + H |
| Copy | ⌘ + C / Ctrl + C |
| Paste | ⌘ + V / Ctrl + V |
| Cut | ⌘ + X / Ctrl + X |
| Delete | Backspace / Delete |
| Clear Selection | Esc |
